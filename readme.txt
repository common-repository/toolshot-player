== Installation ==

= Requires =
* WordPress at least 3.0.1
* PHP at least 5.3

= Instalation steps =

1. Upload the plugin files to the `/wp-content/plugins/toolshot-player` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
